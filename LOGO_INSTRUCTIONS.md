# Logo Setup Instructions

## Step 1: Add Your Logo
1. Save your logo image as `aaryavista-logo.png` 
2. Copy it to the `public` folder in your project
3. The file path should be: `arya-vista-landing-page/public/aaryavista-logo.png`

## Step 2: Logo Specifications
- **Recommended size**: 400x400 pixels or larger (square format works best)
- **Format**: PNG with transparent background (preferred) or JPG
- **File name**: Must be exactly `aaryavista-logo.png`

2. Check that it's in the `public` folder (not `src` or any subfolder)
3. Refresh your browser after adding the file
4. Check browser console for any error messages
